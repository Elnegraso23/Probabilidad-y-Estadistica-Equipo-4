# Stats-Viz-Lab 📈

Una plataforma educativa interactiva y minimalista diseñada para la enseñanza y visualización de conceptos fundamentales de estadística: **Regresión Lineal Simple** y **Diagramas de Dispersión**.

## 🚀 Propósito
Este proyecto fue creado como una herramienta académica para facilitar la comprensión de cómo se relacionan las variables, cómo se calculan los modelos predictivos mediante el método de mínimos cuadrados y cómo interpretar la precisión de los datos a través del coeficiente $R^2$.

## ✨ Características
- **Doble Presentación**: Dos módulos independientes para una enseñanza estructurada.
- **Visualización en Tiempo Real**: Gráficos interactivos generados dinámicamente.
- **Análisis de Correlación**: Explicación visual de tipos de correlación (Positiva, Negativa, Nula).
- **Enfoque Académico**: Diseño editorial limpio, ideal para proyecciones en clase o estudio personal.
- **Navegación Intuitiva**: Soporte para teclado y controles táctiles.

## 🛠️ Tecnologías
- **React 19** & **TypeScript**
- **Vite** (Build Tool)
- **Tailwind CSS** (Styling)
- **Recharts** (Visualización de Datos)
- **Framer Motion** (Animaciones)
- **Lucide React** (Iconografía)

## 📋 Requisitos
- Node.js (versión 18 o superior)
- npm o pnpm

## 🔧 Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/Stats-Viz-Lab.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
